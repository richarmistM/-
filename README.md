# 运筹学与数值优化算法库 (Optimization Algorithms Collection)

本项目包含四个独立的 Python 模块，分别实现了运筹学与数值分析中核心的优化算法，包括单纯形法、对偶单纯形法、梯度下降法以及整数规划的割平面法。

## 📋 目录结构

* **Simplex/**: 标准单纯形法 (Standard Simplex Method)
* **SimDual-Eye/**: 对偶单纯形法 (Dual Simplex Method)
* **tidudown/**: 梯度下降法 (Gradient Descent)
* **yunchouex/**: Gomory 割平面法 (Gomory Cutting Plane Method)

## ⚙️ 环境依赖 (Requirements)

本项目代码基于 Python 开发，推荐使用以下特定版本的依赖库以确保兼容性：

* Python 3.x
* **numpy** == 1.24.4
* **scipy** == 1.10.1
